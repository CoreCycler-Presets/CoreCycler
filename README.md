# CoreCycler Presets
Collection of custom presets, test profiles, guides and optimized configurations for CoreCycler — a popular tool for testing CPU and memory stability under heavy load.

## What's Included
* 📊 Ready-made test presets for different CPU generations and memory configurations
* 📈 Detailed stability testing templates and logging examples
* 🔍 Recommended test durations and settings for daily use versus extreme stress testing
* 🛠️ Custom configuration files for various hardware setups (Intel and AMD)
* 📸 Screenshots, step-by-step guides and result interpretation examples

## How to use
1. Download the archive from the **Releases** tab on the right → select the desired .zip file
2. Extract the files to any convenient folder
3. Run CoreCycler and load the desired preset
4. Choose the appropriate test mode for your hardware
5. Start the test and monitor system stability
6. Analyze results and adjust settings if needed

## Important Notes
- All presets are provided as is — always monitor system temperatures during testing
- Never leave the system unattended during long stress tests
- Create a system restore point before making any changes
- Compatible with Windows 10 and Windows 11
